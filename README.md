# ccc-risk-stratification
Predictive Machine Learning models for risk stratification in Chronic Chagas Cardiomyopathy (CCC). Built via CRISP-DM it features two LightGBM-based models to predict general complication (92% recall) and Sudden Cardiac Death (87% recall) using multimodal and hybrid data and SHAP explainability. MBA in Data Science (USP).
